## [0.1.0] - 2026-03-12

### Added
- Initial release
- Team list view with rankings
- Team detail view with statistics and match history
- Match detail view with score breakdowns
- Search teams by number (press `/`)
- Refresh data (press `r`)
- Support for events with and without rankings
- Response caching with ETag support
- Test mode for viewing historical data
- Proper terminal cleanup on exit

### Features
- Navigate with arrow keys or j/k
- View team statistics (auto, teleop, endgame averages)
- View match schedules (past and future)
- Click through teams from match screens
- Teams sorted by rank (or number if no rankings)
- Unranked teams displayed at bottom sorted by number
